# Deploy (Domain-tár / shared hosting)

1. Töltsd fel a ZIP tartalmát a webtár gyökerébe (példa: public_html / www / htdocs).
2. Győződj meg róla, hogy az `index.html` közvetlenül a webtár gyökerében van.
3. Ellenőrizd, hogy az `assets/` mappa is felkerült (CSS/JS).
4. Ha van gyorsítótár a szolgáltatónál, ürítsd (ha szükséges).

Nincs build lépés. Statikus oldal.


## Képek
A képek az `assets/img/` mappában vannak. Feltöltésnél ezt a mappát is vidd fel.
